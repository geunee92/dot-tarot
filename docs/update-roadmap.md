# DOT TAROT 업데이트 로드맵

---

## v1.0 — 첫 출시

- 오늘의 카드 (데일리 1장)
- 3장 스프레드 (연애/금전/직장)
- AI 해석 (GPT-4o-mini)
- 클래리파이어 카드 (리워드 광고)
- 출석 보상 스킨 시스템 (14/28/42/56일)
- 한국어/영어 지원
- 여정 캘린더 + 히스토리

---

## v1.1 — 리텐션 & 바이럴 (출시 후 1~2주 내)

### 푸시 알림
- 매일 오전 알림: "오늘의 카드가 기다리고 있어요"
- 출석 마일스톤 근접 시: "3일만 더 출석하면 새 스킨 해금!"
- expo-notifications 사용
- 알림 시간 설정 (설정 화면)

### 공유하기
- 링크 카드형 공유 (OG 메타태그)
- GitHub Pages에 공유 전용 웹페이지 생성
- 썸네일 + 제목 + 설명 + 앱스토어 링크
- 카카오톡/iMessage/인스타 등에서 카드형 미리보기

## v1.2 — 콘텐츠 확장 (출시 후 3~4주)

### 스프레드 주제 추가
- 건강
- 학업/시험
- 인간관계

### 추가 스프레드 (Follow-up Reading)
- 기존 3장 스프레드 결과에서 "더 깊이 알아보기" 옵션
- 새 질문 입력 + 3장 추가 뽑기
- 이전에 뽑은 카드는 덱에서 제외
- AI가 첫 번째 리딩 + 두 번째 리딩을 종합 해석
- 리워드 광고 시청 후 해금

## v1.3 — 홈화면 & 참여 (출시 후 1~2개월)

### 메모/일기 기능 강화

## v1.4 — 수익화 & 소셜 (출시 후 2~3개월)

### 프리미엄 스킨 팩
