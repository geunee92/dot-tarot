## 캡처할 화면 (5개)

### 1. Daily Card 결과 ⭐ (가장 중요)
- [ ] 카드가 앞면으로 뒤집힌 상태
- [ ] 예쁜 카드 선택 (The Sun, The Star, The Fool 등)
- [ ] 정방향(Upright) 상태 권장

**캡처 시점**: 카드 뒤집기 완료 후

### 2. 픽셀 아트 카드 강조
- [ ] 카드 상세 화면 또는
- [ ] 카드 뒤집기 중간 상태 또는
- [ ] 여러 카드가 보이는 화면

**목적**: 픽셀 아트 스타일 차별화 강조

### 3. Spreads 주제 선택
- [ ] SpreadsScreen 진입
- [ ] 연애/금전/직장 3개 카드 모두 보이게
- [ ] 아직 선택하지 않은 깔끔한 상태

**캡처 시점**: Spreads 탭 진입 직후

### 4. 3-Card 스프레드 결과
- [ ] 3장 카드가 모두 펼쳐진 상태
- [ ] 흐름-영향-조언 레이블 보이게
- [ ] 해석 텍스트 일부 보이면 좋음

**캡처 시점**: 스프레드 완료 후

### 5. Journey 캘린더
- [ ] JourneyScreen 캘린더 뷰
- [ ] 몇 개 날짜에 리딩 기록 표시되면 좋음
- [ ] (없으면 오늘 날짜만 표시된 상태도 OK)

**캡처 시점**: Journey 탭 진입

## 캡처 팁

### 베스트 프랙티스
1. **밝은 카드 선택**: The Sun, The Star, The World 등
2. **정방향 카드**: 긍정적인 이미지
3. **화면 정리**: 알림 배지, 시간 등 깔끔하게

### iPhone 스크린샷 방법
- **Face ID iPhone**: 전원 + 볼륨 업 동시 누르기
- **홈 버튼 iPhone**: 전원 + 홈 버튼 동시 누르기

### Mac으로 전송
1. AirDrop (가장 빠름)
2. iCloud Photos
3. 케이블 연결 후 이미지 캡처 앱

## 캡처 후 처리 (선택사항)

### A. 그대로 업로드
- App Store Connect에서 iPhone 스크린샷 그대로 업로드 가능
- 텍스트 오버레이 없이도 심사 통과 가능

### B. 텍스트 오버레이 추가 (권장)
Figma 또는 Canva에서:
1. 1290 x 2796 캔버스 생성
2. 배경색: #0f0f23
3. iPhone 목업에 스크린샷 삽입
4. 헤드라인 텍스트 추가

**헤드라인 예시**:
| 화면 | 한국어 | English |
|------|--------|---------|
| Daily Card | 매일 무료로 뽑는 오늘의 타로 | Your Free Daily Tarot Card |
| Pixel Art | 픽셀 아트로 만나는 타로 카드 | Pixel Art Tarot Cards |
| Spreads | 연애, 금전, 직장 운세 리딩 | Love, Money & Career |
| 3-Card | 3장으로 읽는 오늘의 에너지 | 3-Card Energy Reading |
| Journey | 나의 타로 여정 기록 | Track Your Journey |
