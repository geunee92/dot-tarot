## 기능 테스트 체크리스트

### 1. 오늘의 카드 (Daily Card)

| 테스트 | 예상 결과 | 확인 |
|--------|----------|------|
| [ ] 첫 방문 시 카드 뽑기 가능 | 무료로 뽑기 가능 | |
| [ ] 카드 뒤집기 애니메이션 | 부드럽게 뒤집힘 | |
| [ ] 정방향/역방향 랜덤 | 두 방향 모두 나옴 | |
| [ ] 카드 해석 표시 | 키워드, 의미, 부적 표시 | |
| [ ] 메모 저장 | 메모 입력 후 저장됨 | |
| [ ] 두 번째 뽑기 시도 | "오늘 이미 뽑았습니다" 또는 광고 시청 옵션 | |
| [ ] 자정 후 리셋 | 다음 날 다시 무료 뽑기 가능 | |

### 2. 3장 스프레드 (Spreads)

| 테스트 | 예상 결과 | 확인 |
|--------|----------|------|
| [ ] 연애 주제 선택 | 연애 스프레드 시작 | |
| [ ] 금전 주제 선택 | 금전 스프레드 시작 | |
| [ ] 직장 주제 선택 | 직장 스프레드 시작 | |
| [ ] 3장 카드 배치 | 흐름-영향-조언 순서로 배치 | |
| [ ] 카드 뒤집기 | 각 카드 터치 시 뒤집힘 | |
| [ ] 에너지 패턴 분석 | 패턴 텍스트 표시 | |
| [ ] 첫 번째 무료 | 각 주제 첫 번째는 무료 | |
| [ ] 두 번째 시도 | 광고 시청 후 가능 | |

### 3. 클래리파이어 카드 (Clarifier)

| 테스트 | 예상 결과 | 확인 |

### 4. 여정 (Journey)

| 테스트 | 예상 결과 | 확인 |

### 5. 설정 (Settings)
